About This Prototype

This is a standalone, installable, offline-first prototype intended for real trainer feedback.

Version 6 GitHub-Ready Offline PWA

JSON is now the manifest/index. Dog photos and audio cues are stored as real files using OPFS where available. On iPhone/iOS or restrictive local servers, the app safely falls back to IndexedDB/localStorage media storage, then exports everything inside ZIP packs under media/photos/ and media/audio/. The app includes a web app manifest and service worker so it can be installed and used offline after first load.

This keeps the app scalable, avoids huge base64 JSON files, improves playback reliability, and aligns with future encrypted App Library payloads.
